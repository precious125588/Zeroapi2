Authentication

All API endpoints (except search and public pages) require an API key. Pass it in the header:

Header
Query
X-API-Key: YOUR_API_KEY
GET /api/tiktok?url=...&apikey=YOUR_API_KEY

Registration (Email OTP)

Step 1 — Send OTP

POST/auth/send-otp
FieldRequiredDescription
emailYesGmail or any email address
purposeNoregister (default) or login
JavaScript
cURL
// Step 1: request OTP const r1 = await fetch('/auth/send-otp', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email: 'you@gmail.com', purpose: 'register' }) }); const d1 = await r1.json(); // d1.smtp_enabled = true if email was sent // d1.dev_code = code shown only when SMTP not configured
curl -X POST /auth/send-otp \ -H "Content-Type: application/json" \ -d '{"email":"you@gmail.com","purpose":"register"}'

Step 2 — Register with OTP

POST/auth/register
JavaScript
// Step 2: complete registration const r2 = await fetch('/auth/register', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: 'John Doe', email: 'you@gmail.com', password: 'yourpassword', code: '123456' // OTP from email or dev_code }) }); const d2 = await r2.json(); // d2.data.api_key — your API key (save this!) // d2.data.token — session token // d2.data.limit — "40 requests/day"

Login

POST/auth/login
JavaScript
const res = await fetch('/auth/login', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email: 'you@gmail.com', password: 'yourpassword' }) }); const { data } = await res.json(); // data.api_key — use in X-API-Key header // data.token — use in X-Session-Token header // data.used — requests used today // data.limit — daily limit

Response Format

{ "success": true, "cached": false, "data": { "platform": "tiktok", "title": "Video title", "quality": "hd", "author": "username", "thumbnail": "https://...", "media": [ { "type": "video", // "video" | "audio" | "image" | "file" "url": "https://...", // Direct download URL "quality": "hd", // "4k" | "1080p" | "720p" | "hd" | "sd" | "audio" "container": "mp4", // File extension "note": "no watermark" // Optional info } ] } }

TikTok

GET/api/tiktok?url=VIDEO_URL

Returns HD video (no watermark), SD video, audio, and images for slideshows.

JavaScript
Python
cURL
const res = await fetch('/api/tiktok?url=https://www.tiktok.com/@user/video/123456', { headers: { 'X-API-Key': 'YOUR_API_KEY' } }); const { data } = await res.json(); const hdVideo = data.media.find(m => m.type === 'video' && m.quality === 'hd'); // hdVideo.url = direct MP4 (no watermark)
import requests r = requests.get('/api/tiktok', params={'url': 'https://www.tiktok.com/@user/video/123456'}, headers={'X-API-Key': 'YOUR_API_KEY'}) data = r.json()['data'] video_url = next(m['url'] for m in data['media'] if m['type'] == 'video')
curl -H "X-API-Key: YOUR_API_KEY" \ "/api/tiktok?url=https://www.tiktok.com/@user/video/123456"

Movies & TV NEW

GET/api/movies?url=TMDB_ID&type=movie
ParamRequiredExample
urlYesTMDB ID (550), TMDB URL, or IMDB tt-id
typeNomovie (default) or tv
seasonNo1 (for TV series)
episodeNo1 (for TV series)
⚠ Returns HLS (.m3u8) streams and embed player URLs. Use HLS.js or VLC for m3u8 streams, or use the embed URLs in an iframe.
Movie
TV Episode
// Movie by TMDB ID const res = await fetch('/api/movies?url=550&type=movie', { headers: { 'X-API-Key': 'YOUR_API_KEY' } }); const { data } = await res.json(); // data.media[] contains HLS streams and embed URLs const hls = data.media.find(m => m.container === 'm3u8'); const embed = data.media.find(m => m.container === 'embed'); // Use hls.url in HLS.js player, or embed.url in an iframe
// TV show episode (Breaking Bad, S1E1) const res = await fetch('/api/movies?url=1396&type=tv&season=1&episode=1', { headers: { 'X-API-Key': 'YOUR_API_KEY' } }); const { data } = await res.json(); // data.subtitles[] — subtitle tracks with lang and url

Error Codes

StatusMeaning
401Missing or invalid API key
403Key disabled, blacklisted, or IP blocked
429Daily request limit reached — resets each day
422No downloadable media found (private/removed content)
502Provider failed — try again or use another platform
503Endpoint disabled or in maintenance mode
